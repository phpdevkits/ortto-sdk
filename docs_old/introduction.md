# Introduction

## What is Ortto SDK?

Ortto SDK is a modern, elegant Laravel package that provides a fluent interface to interact with the [Ortto Customer Data Platform API](https://ortto.com). Built on top of [Saloon](https://docs.saloon.dev/), it takes the pain and hassle out of integrating Ortto's powerful marketing automation and customer data features into your Laravel applications.

## Why Ortto SDK?

Ortto SDK simplifies the complexity of API integration by providing:

- **Type-safe requests and responses** - Leverage PHP 8.4+ features for robust, reliable code
- **Fluent, expressive syntax** - Write clean, readable code that clearly expresses your intent
- **Laravel-first design** - Seamless integration with facades, service providers, and configuration
- **Comprehensive testing support** - Built-in support for mocking and architecture testing with [Lawman](https://github.com/JonPurvis/lawman)
- **Production-ready** - Rate limiting, retry logic, and error handling built-in

## What Can You Do?

With Ortto SDK, you can easily:

- **Manage People** - Create, update, retrieve, and delete contacts in your Ortto CDP
- **Track Activities** - Record custom events and user behaviors for segmentation
- **Send Campaigns** - Trigger email campaigns and marketing automation workflows
- **Manage Tags** - Organize and segment your audience with tags and custom fields
- **Query Data** - Retrieve customer data and activity history for analytics

## Requirements

Before you begin, make sure your system meets these requirements:

- PHP 8.4 or higher
- Laravel 10.x or 11.x
- An active [Ortto account](https://ortto.com) with API access
